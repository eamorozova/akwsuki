# sledilo

Веб-сервис для сравнения конфигурационных переменных между двумя точками
(ветка + окружение) внутри одного git-репозитория функционального продукта (ФП).

## Структура

- `server/` — бэкенд (Fastify, TypeScript). Ядро сравнения: `scan` → `diff` → `compare`.
- `web/` — фронтенд (React + Vite).

## Разработка

```bash
npm install         # установить зависимости (workspaces)
npm test            # юнит-тесты ядра (vitest)
npm run typecheck   # проверка типов
```
